# Code Repository Setup

## 1. VERIFY Repository
- Ensure Git repository has been initialized locally or on GitHub.
- Verify the `origin` remote is correctly configured and user has appropriate push/pull access.

## 2. Setup Git Hooks with Husky and lint-staged
- Install [Husky](https://typicode.github.io/husky/#/) to enable Git hooks
- Install [lint-staged](https://github.com/okonet/lint-staged) to optimize checks
- Configure the `pre-commit` hook to:
  - Run `eslint` and `tsc --noEmit` **on staged files only** using `lint-staged`
  - Block commit if linting or type checking fails
